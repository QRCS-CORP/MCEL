\doxysubsection{Detailed Description}
The MCEL anchor reference container. 

This structure describes an external witness reference (e.\+g. chain id and transaction id). The {\ttfamily chain\+\_\+id} and {\ttfamily reference} fields point to caller-\/owned memory.
